A modern video transcoder for the modern day needs.

Download GitHub

Elegant and performant

Frontend built with HTML and JavaScript for optimal performance and a seamless user experience, but the actual heavy lifting is done by your favorite CLI tools like FFmpeg, x264, qAAC and more.

Efficient and useful presets

The default presets are optimized for various use cases, with minimal configuration. From compatibility with older devices to the most modern standards, Zendemux has you covered. And if you require more control, you can set advanced parameters and share those presets with others.
